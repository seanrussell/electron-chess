Welcome!

With Electron Chess, you can load PGN chess games and also start your own games.

  • folder_open Open Game

    Cmd (or Ctrl) + O
    From the Electron menu, select Open Game to navigate to your existing PGN chess game.

  • create_new_folder New Game

    Cmd (or Ctrl) + N
    From the Electron menu, select New Game to begin a new chess game.

  • save Save Game

    Cmd (or Ctrl) + S
    From the Electron menu, select Save Game to save a game you started in PGN format.

  • close Close Game

    Cmd (or Ctrl) + C
    From the Electron menu, select Close Game to close the game board and return to this screen.
